
# Products Service

## Lancer en local
1. Installer MongoDB
2. Lancer `npm install`
3. Lancer le service : `npm run dev`
4. Swagger : http://localhost:3002/api-docs

## Variables d'environnement
- `PORT` : port du service (default 3002)
- `MONGO_URI` : URI MongoDB (default `mongodb://localhost:27017/products_db`)
